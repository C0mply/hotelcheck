# HotelCheck — Hotel Reservation Management System

## Product Pitch
HotelCheck is a desktop application designed to help small hotels manage rooms, guests, and reservations in one place.

## Problem
Small hotels often rely on scattered notes, spreadsheets, or manual processes to track room availability, guest information, and bookings. This can lead to errors, double-booking risks, and inefficient administration.

## Target Users
The main users of this system are hotel administrators, reception staff, and small accommodation businesses that need a simple internal booking and management tool.

## Core Features
- Room management
  - Add, edit, and delete hotel rooms
  - Store room number, type, capacity, price, and availability

- Guest management
  - Add, edit, and delete guest records
  - Store guest name, surname, phone number, and document information

- Reservation management
  - Create and delete reservations
  - Link reservations to rooms and guests
  - Store check-in and check-out dates

- Data persistence
  - Save and load application data using a local JSON file

## Project Goal
The goal of this project is to build a practical hotel reservation management application during the course and use it as the basis for further development and the final course report.
